# Steuerung
P -> "pausieren/fortfahren" (beim start ist das Spiel standardgemäß pausiert)
W -> vorwärts
S -> rückwärts
A -> links
D -> rechts
W + L_SHIFT -> sprint
W + L_CTRL -> schleichen
SPACE -> springen

# Interaktionen
Man kann mit Münzen interagieren, indem man mit denen kollidiert. Mit der Interaktion sammelt man die Münze ein.
